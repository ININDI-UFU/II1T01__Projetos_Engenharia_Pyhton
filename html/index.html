04 / 15

Formulação do Problema

CONVERSÃO RMS → PICO

Para obter o valor instantâneo máximo:

Vpico = VRMS × √2

Para 220 V RMS:

Vpico ≈ 311,13 V

CIRCUITO CONDICIONADOR

A rede resistiva deve:

  • ▸ Atenuar o sinal CA de ±311 V para ±2,5 V
  • ▸ Adicionar offset CC de +2,5 V
  • ▸ Resultado: saída entre 0 V e 5 V
  • ▸ Usar apenas resistores comerciais (R1, R2, R3)
⚠️O sinal de entrada possui semiciclos positivo (+311 V) e negativo (−311 V) — ambos devem ser mapeados para 0–5 V
05 / 15

Esquemático do Circuito Condicionador

FUNÇÃO DOS COMPONENTES

R3 (Série)

Atenua o sinal CA. Forma divisor de tensão com R1‖R2.

R1 (para Vcc)

Introduz offset CC positivo. Eleva o sinal para faixa 0-5V.

R2 (para GND)

Completa divisor de tensão. R2 = R1 simplifica o cálculo.

Vout

Vcc + ganho_CA × Vpico. Deve ficar entre 0V e 5V.

06 / 15

Derivação Matemática do Circuito

1ETAPA 1: CONVERSÃO RMS → PICO

O valor RMS é o valor eficaz da tensão CA senoidal. Para obter o valor de pico:

Vpico = VRMS × √2

220 × 1,4142 ≈ 311,13 V

2ETAPA 2: GANHO CA — DIVISOR DE TENSÃO

Para o sinal CA, Vcc é curto-circuito (fonte ideal). R1 e R2 ficam em paralelo:

R1‖R2 = (R1 × R2) / (R1 + R2)

O ganho CA real do divisor R3 + (R1‖R2):

GCA = R1‖R2 / (R3 + R1‖R2)
3ETAPA 3: OFFSET CC — NÍVEL DC

Para CC, a entrada CA é zero. O offset vem de Vcc através de R1. R3 e R2 ficam em paralelo vistos de R1:

VCC = [R3‖R2 / (R1 + R3‖R2)] × Vcc

Este nível DC desloca a senoide para que o vale fique próximo de 0V.

4ETAPA 4: SAÍDA FINAL COMBINADA

Superposição: somamos a componente CA atenuada com o offset CC:

Vout = VCC + GCA × Vpico

Critério: 0 ≤ Vout ≤ 5V para compatibilidade com ADC.

SÉRIE E24 — COMO FUNCIONA

  • 23 valores base de 10 a 82 (espaçamento logarítmico)
  • 7 décadas de ×1 até ×1M (multiplicadores)
  • Total: 23 × 7 = 161 valores disponíveis para busca
  • Tolerância típica: ±5%

POR QUE SÉRIE COMERCIAL?

  • Resistências com valores arbitrários não existem no mercado
  • A busca garante que o circuito pode ser montado com peças reais
  • Valores próximos são cobertos pela tolerância do componente

        43, 47, 51, 56, 62,

                  for d in decadas]
